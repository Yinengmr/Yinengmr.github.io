Linux 简介

Linux 内核最初只是由芬兰人林纳斯·托瓦兹（Linus Torvalds）在赫尔辛基大学上学时出于个人爱好而编写的。

Linux 是一套免费使用和自由传播的类 Unix 操作系统，是一个基于 POSIX 和 UNIX 的多用户、多任务、支持多线程和多 CPU 的操作系统。

Linux 能运行主要的 UNIX 工具软件、应用程序和网络协议。它支持 32 位和 64 位硬件。Linux 继承了 Unix 以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。

GIT ssh-key 本地秘钥生成

2018-05-10 #github
一、设置git的username 和 useremail git config --global user.name &quot;user.name&quot; git config --global user.email &quot;user.email&quot; 二、生成秘钥 1）查看是否有ssh秘钥 ls ~/.ssh id_rsa id_rsa.pub known_hosts 2）没有id_rsa 和 id_rsa.pub则生成 ssh-keygen -t rsa -C &quot;your_email@youremail.com&quot; 执行完ssh-keygen之后会在，用户目录下的.ssh文件下，生成一个id_rsa文件和id_rsa.pub文件。 id_rsa文件是私钥，要保存好，放在本地，私钥可以生产公钥，反之不行。 id_rsa.pub文件是公钥，可以用于发送到其他服务器，或者git上 3） 设置ssh无密码访问git仓库 cat id_rsa.pub 复制字符串 到网站添加 以github为例，找到个人主页，点击[settings],找到[SSH and GPG keys] ，新建SSH keys,将本地id_rsa.pub的内容复制到key里面，tittle可以随便填写，这样就配置好了。
